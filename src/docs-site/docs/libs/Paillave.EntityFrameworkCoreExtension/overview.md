This library provides connectors and helpers related to **EntityFrameworkCore Extension**.

## What it provides

- Connectors and helpers specific to EntityFrameworkCore Extension.
- Integration points that fit into the core ETL pipeline.

## When to use it

- You need to read from, write to, or interact with EntityFrameworkCore Extension within an ETL flow.
