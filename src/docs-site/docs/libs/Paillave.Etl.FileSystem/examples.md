## Read a folder and write new files

```csharp
using Paillave.Etl.Core;
using Paillave.Etl.FileSystem;

var runner = StreamProcessRunner.Create<string>(s => s
    .CrossApplyFolderFiles("list files", "*.txt", recursive: true)
    .Do("print", f => Console.WriteLine(f.Name))
    .Select("create output", _ => FileValueWriter
        .Create("summary.txt")
        .WriteLine("Generated by ETL"))
    .WriteToFile("write output", f => $"/tmp/{f.Name}"));

await runner.ExecuteAsync("/data/input", new ExecutionOptions<string>());
```
